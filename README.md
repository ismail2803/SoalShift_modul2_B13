#### Daemon
```
pid_t pid;
pid = fork();

if (pid < 0) {
  exit(EXIT_FAILURE);
}

if (pid > 0) {

  // simpan PID dari child proces

  exit(EXIT_SUCCESS);

```

```

```

  exit(EXIT_FAILURE);
}
```
setsid() dijalankan agar child process memiliki SID unik yang berasal dari kernel agar prosesnya dapat berjalan.
```
